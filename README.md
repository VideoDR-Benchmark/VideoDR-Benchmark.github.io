## Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run serve

# Build for production
npm run build
```

### Development

The application will be available at `http://localhost:8080`
